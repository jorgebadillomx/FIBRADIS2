# Historia 4.5.2: Scraping del cuerpo completo del artículo

Status: cancelled

## Decisión de diseño

Cancelada antes de implementar. Razón: el equipo decidió no scrapear el cuerpo completo de los artículos de terceros para evitar riesgos legales de copyright (republicar contenido completo de El Financiero, Expansión, etc.) y deuda técnica operacional (scrapers que se rompen con cambios de estructura HTML).

**Alternativa adoptada:** La historia 4.5.3 muestra el **resumen de Gemini** (ya disponible desde 4.3) + og:image como preview. El botón "Leer más" lleva siempre a la fuente original. Ver historia 4.5.3 para el diseño final.

No se crean archivos. No se implementa nada de esta historia.
